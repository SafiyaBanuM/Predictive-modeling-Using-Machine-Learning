# Predictive Modeling Using Machine Learning

##  Project Overview
This project demonstrates supervised machine learning using the Titanic dataset. A Decision Tree Classifier is trained to predict whether a passenger survived based on features such as age, gender, passenger class, fare, and family information.

##  Objective
Build a predictive model to classify passenger survival and evaluate its performance.

##  Dataset
- Titanic Dataset
- 891 records
- 12 features

##  Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

##  Machine Learning Algorithm
- Decision Tree Classifier

##  Steps Performed
- Data Loading
- Data Cleaning
- Handling Missing Values
- Encoding Categorical Variables
- Feature Selection
- Train-Test Split
- Model Training
- Prediction
- Model Evaluation

##  Results
- Model Accuracy: **78.77%**
- Evaluation using Confusion Matrix

##  Output
The project generates:
- Accuracy Score
- Confusion Matrix

##  How to Run

```bash
pip install pandas matplotlib scikit-learn
python Predictive-Modeling-Using-Machine-Learning.py
```

##  Outcome
Successfully developed and evaluated a supervised machine learning model using the Decision Tree algorithm.
